# cv2pdf

Convert a Markdown CV to PDF.

## Install:

Requirements:

 * Node.js

```
$ npm install -g cv2pdf
```

## Usage

```
# simple example
$ cv2pdf cv.md

# provide a filename to the output file
$ cv2pdf --out=renato-martins-cv.pdf cv.md

# custom css
$ cv2pdf --css=style.css cv.md

# usage help
$ cv2pdf --help
```
